# PEPTQ Temp Print Site

[![Deploy Vite Site to GitHub Pages](https://github.com/staylegitlonewolf/peptq-temp-print-site/actions/workflows/deploy-pages.yml/badge.svg)](https://github.com/staylegitlonewolf/peptq-temp-print-site/actions/workflows/deploy-pages.yml)

OL1735 canvas editor and print workflow app built with React + Vite.

## Live Site

- https://staylegitlonewolf.github.io/peptq-temp-print-site/

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Deploy

This repo deploys automatically to GitHub Pages from GitHub Actions on every push to `main`.

Quick deploy commands:

```bash
cd C:\Users\Lonewolf\Desktop\PrintCenterPage_TEMPLATE
git add .
git commit -m "fix: your message"
git push
```

## Notes

- GitHub Pages source should remain set to `GitHub Actions`.
- If a deploy fails, check the Actions tab and share the error to patch quickly.
